5.2 Inizializzazione e Caricamento Plugin
  • wp-settings.php è il file che orchestra tutto.
  • 1. Carica le librerie, imposta le variabili di base.
  • 2. Carica i plugin "Must-Use" (mu-plugins).
  • 3. Poi carica i plugin attivi (dalla tabella wp_options).
  • 4. Infine, carica il functions.php del tema.
Approfondimento

L'ordine di caricamento è importante. I plugin "Must-Use" vengono eseguiti per primi, il che li rende un buon posto per mettere codice di sicurezza critico. Poi vengono i plugin normali, e solo alla fine il tema.

Se un plugin e un tema definiscono una funzione con lo stesso nome, questo può causare un errore fatale. La conoscenza di questo ordine è essenziale per il debugging.
