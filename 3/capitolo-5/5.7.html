5.7 REST API Lifecycle (`/wp-json`)
  • Le richieste a /wp-json/ hanno un lifecycle diverso.
  • WordPress rileva la richiesta e carica il server REST API.
  • Viene eseguito l'hook rest_api_init, where custom routes are registered.
  • L'autenticazione è gestita separatamente.
  • ⚠️ Molti plugin di sicurezza "tradizionali" non proteggevano adeguatamente la REST API.
Approfondimento

La REST API è quasi un'applicazione a sé stante dentro WordPress. Ha i suoi hooks, il suo sistema di routing e di autenticazione. Quando è stata introdotta, ha aperto una nuova, vasta superficie di attacco.

Molti siti avevano regole di sicurezza che si applicavano solo al frontend "classico", lasciando gli endpoint API esposti. È fondamentale assicurarsi che le protezioni coprano anche questo percorso, inclusi rate limiting e validazione input.
