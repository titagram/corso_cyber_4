5.1 Flusso di Caricamento Iniziale
  • Tutto parte da index.php.
  • index.php → wp-blog-header.php
  • wp-blog-header.php → wp-load.php
  • wp-load.php carica wp-config.php e poi wp-settings.php, where the magic begins.
Approfondimento

Capire il flusso di bootstrap è importante per sapere dove intervenire. Ogni richiesta a una pagina WordPress passa attraverso questa catena di file.

Se un attaccante modifica uno di questi file core, può eseguire codice malevolo all'inizio di ogni singola richiesta, prima ancora che i plugin di sicurezza vengano caricati. Questo è il motivo per cui i file core devono essere read-only per www-data.
