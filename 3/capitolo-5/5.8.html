5.8 Come intercettare la richiesta
  • Per bloccare una richiesta malevola il prima possibile, bisogna usare un hook precoce.
  • Un mu-plugin che si aggancia a muplugins_loaded.
  • Dentro la funzione, si analizza $_SERVER, $_GET, $_POST.
  • Se si rileva una minaccia:
wp_die('Richiesta bloccata.');
Approfondimento

Se volete scrivere il vostro piccolo sistema di blocco, questo è il modo. Create un file in wp-content/mu-plugins. Dentro questo file, usate un hook che viene eseguito molto presto.

Analizzate le variabili della richiesta alla ricerca di pattern sospetti. Se trovate qualcosa, usate wp_die() per fermare tutto. È un modo semplice ma efficace per creare regole di blocco personalizzate senza installare un plugin pesante.
