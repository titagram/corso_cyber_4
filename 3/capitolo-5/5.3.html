5.3 La Main Query (WP_Query)
  • Dopo aver caricato tutto, WordPress analizza l'URL.
  • Crea un oggetto WP_Query che traduce l'URL in una query SQL.
  • L'hook pre_get_posts permette di modificare questa query prima che venga eseguita.
  • È qui che la richiesta "diventa" una query al database.
Approfondimento

La WP_Query è il cuore del sistema di recupero dei contenuti. È un pezzo di codice molto complesso che costruisce dinamicamente una query SQL basata sui parametri dell'URL.

L'hook pre_get_posts è estremamente potente. I plugin lo usano per creare filtri di ricerca avanzati o per escludere certe categorie di post, ma se usato male, può rompere completamente il sito o introdurre vulnerabilità SQLi.
