4.3 La tabella `wp_users`
  • Contiene la lista degli utenti.
  • user_login: il nome utente.
  • user_pass: l'hash della password.
  • user_email: l'indirizzo email.
  • Se un attaccante ottiene un dump di questa tabella, può tentare di crackare le password offline.
Approfondimento

La tabella wp_users è un obiettivo ovvio. Contiene le informazioni di login. Anche se le password sono hashate, un dump del database permette a un attaccante di lanciare un attacco a dizionario o brute-force offline, senza limiti di tentativi.

Se le password sono deboli, verranno trovate in poco tempo. Questo è il motivo per cui un backup del database deve essere protetto con la stessa cura del database stesso.
