3.4 `wp-includes`: Il Core Intoccabile
  • Contiene le funzioni e le classi fondamentali di WordPress.
  • Questa cartella non dovrebbe MAI essere scrivibile da www-data.
  • Una modifica a un file qui dentro può creare una backdoor a livello core.
  • Strumenti come WordFence scansionano l'integrità di questi file.
Approfondimento

Se un attaccante riesce a modificare un file in wp-includes, la compromissione è profonda. Può inserire una backdoor in una funzione usata da tutto il sito, rendendola quasi invisibile.

Per questo, i permessi su questa cartella devono essere restrittivi. L'utente www-data deve solo poter leggere. I plugin di sicurezza controllano le somme hash di questi file per rilevare modifiche non autorizzate, confrontandole con quelle dei file originali di WordPress.
