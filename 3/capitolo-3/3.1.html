3.1 Anatomia della Root Directory
  • wp-admin/: Il pannello di amministrazione.
  • wp-includes/: File e librerie del core di WP.
  • wp-content/: Il "regno" degli utenti: temi, plugin, uploads.
  • wp-config.php: Il file più critico. Contiene le credenziali del DB.
Approfondimento

La struttura di WordPress è molto chiara. wp-admin e wp-includes sono il "core". In un setup ideale, l'utente www-data non dovrebbe mai poterli modificare.

wp-content è dove avviene tutta l'azione: i temi, i plugin che installiamo, e i file che carichiamo finiscono tutti qui. È la superficie di attacco principale, perché è l'unica parte che, per design, deve essere parzialmente scrivibile dal web server.
