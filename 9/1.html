CVE-2021-44228: Quando i Log prendono vita

Log4Shell: Il "Mostro"
Dati

Data: Dicembre 2021 (Zero-Day).

CVSS: 10.0 (Criticità Assoluta).

Vettore: JNDI Injection via Log4j.

Effetto: RCE (Esecuzione Codice Remoto) senza autenticazione.

Spiegazione

Cos'è Log4j? È la libreria standard usata in Java per scrivere i file di log. È onnipresente: da Minecraft ai server bancari, da iCloud ai tostapane smart.

Il Paradosso: Una funzionalità pensata per rendere i log "più intelligenti" (arricchendoli con dati esterni) ha permesso agli attaccanti di trasformare una semplice riga di testo in un comando di sistema.
