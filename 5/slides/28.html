Slide 4.5: Exploitation con Metasploit
Teoria: Metasploit Framework

Metasploit è il framework di exploitation più usato al mondo. Offre:


  • ~2000 exploit pronti all'uso
  • Payload avanzati (Meterpreter)
  • Post-exploitation automatizzata
  • Gestione sessioni multiple
  • Integrazione con database

Componenti principali:

  • exploit/ - Moduli di attacco
  • payload/ - Codice da eseguire
  • auxiliary/ - Scanner, fuzzer
  • post/ - Post-exploitation

Security Insight: Metasploit è potentissimo ma "rumoroso". Molti IDS/AV riconoscono i suoi payload. Per operazioni stealth, preferite exploit custom.
