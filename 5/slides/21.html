Capitolo 3: Vulnerability Assessment & Triage

Slide 3.4: CVE Research e CVSS Scoring
Teoria: Capire la Gravità

Il CVSS (Common Vulnerability Scoring System) è lo standard per valutare la gravità delle vulnerabilità.


Scala CVSS v3.1:

0.0 None
0.1-3.9 Low
4.0-6.9 Medium
7.0-8.9 High
9.0-10.0 Critical

Security Insight: Un CVSS alto non significa automaticamente "facile da sfruttare". Leggete sempre i dettagli: Attack Vector, Privileges Required, User Interaction.

Lab: Ricerca CVE Approfondita

Risorse per la ricerca CVE:

• https://nvd.nist.gov/ (NVD - Database ufficiale) • https://cve.mitre.org/ (MITRE - Autorità CVE) • https://wpscan.com/vulnerability/ (WP specifico) • https://www.exploit-db.com/ (Exploit-DB) • https://cvedetails.com/ (Statistiche) • https://vulners.com/ (Aggregatore)

Esempio CVE-2020-36385:

CVE ID: CVE-2020-36385 CVSS: 9.8 (Critical) Attack Vector: Network Attack Complexity: Low Privileges Required: None User Interaction: None Description: Simple File List before 4.2.3 allows unauthenticated file upload and remote code execution via crafted request.

⚠️ CVSS 9.8 + No Auth Required = Priorità MASSIMA
