Capitolo 4: Exploitation

Slide 4.8: Stabilizzare e Migliorare la Shell
Teoria: Shell Upgrade

Una reverse shell "raw" è instabile: no tab completion, no CTRL+C, no editor. Dobbiamo stabilizzarla.


Problemi della shell raw:

  • CTRL+C chiude la connessione
  • No history dei comandi
  • No autocompletamento
  • Alcuni comandi non funzionano
  • Output non formattato

Obiettivo: Ottenere una shell PTY (pseudo-terminal) completa, simile a SSH.


Security Insight: Una shell stabile è essenziale per il post-exploitation. Molti tool (linpeas, pspy) richiedono un terminale completo.

Lab: Python PTY Upgrade

Step 1: Spawn PTY con Python

www-data@target:$ python3 -c 'import pty;pty.spawn("/bin/bash")' www-data@target:$

Step 2: Background e configurazione

www-data@target:$ ^Z # CTRL+Z per background [Sul tuo Kali:] └─$ stty raw -echo; fg [Torna sulla shell target:] www-data@target:$ export TERM=xterm www-data@target:$ export SHELL=/bin/bash www-data@target:$ stty rows 40 cols 160

Risultato:

✓ Tab completion funziona ✓ CTRL+C non chiude la shell ✓ Vim, nano, less funzionano ✓ Clear screen funziona
