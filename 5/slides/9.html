Capitolo 1: Network Discovery & Fingerprinting

Slide 1.8: Identificazione Technology Stack Completo
Teoria: Stack Fingerprinting Avanzato

Oltre a web server e CMS, vogliamo mappare l'intero stack tecnologico: librerie JavaScript, CDN, analytics, framework CSS.


Tecnologie da identificare:

  • CMS e versione (WordPress 5.x)
  • Framework JS (jQuery, React, Vue)
  • Librerie vulnerabili (jQuery < 3.5.0)
  • CDN utilizzato (Cloudflare, AWS)
  • Analytics (Google, Matomo)
  • Plugin di caching (W3 Total Cache)

Security Insight: Librerie JavaScript obsolete (es. jQuery 1.x) possono essere vettori per XSS. Strumenti come Retire.js le identificano automaticamente.

Lab: Whatweb e WPScan
