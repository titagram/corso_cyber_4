Capitolo 3: Vulnerability Assessment & Triage

Slide 3.3: Ricerca Exploit con Searchsploit
Teoria: Dall'Informazione all'Azione

Searchsploit è l'interfaccia CLI per Exploit-DB, il più grande archivio pubblico di exploit e PoC.


Cosa contiene Exploit-DB:

  • Script Python, Ruby, Bash
  • Moduli Metasploit
  • Proof of Concept (PoC)
  • Documentazione vulnerabilità
  • Shellcode e payload

Comandi utili:

  • searchsploit -m [ID] - Copia exploit
  • searchsploit -x [ID] - Visualizza exploit
  • searchsploit -u - Aggiorna database

Security Insight: LEGGETE sempre l'exploit prima di eseguirlo! Potrebbe contenere backdoor o richiedere modifiche per funzionare.

Lab: Trovare Exploit

Ricerca per nome plugin:

┌──(kali㉿kali)-[~] └─$ searchsploit wordpress simple file list

Output:

------------------------------------------------- Exploit Title | Path ------------------------------------------------- WordPress Plugin Simple File List 4.2.2 | php/webapps/49253.py - Unauthenticated Arbitrary File Upload WordPress Plugin Simple File List < 6.0 | php/webapps/51668.py - Remote Code Execution -------------------------------------------------

Copiare l'exploit nella directory corrente:

└─$ searchsploit -m 49253 Copied to: /home/kali/49253.py

Analizzare l'exploit:

└─$ head -50 49253.py
