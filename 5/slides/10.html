Capitolo 2: WordPress Enumeration

Slide 2.1: Passive Scan con WPScan
Teoria: Mappare senza far Rumore

WPScan è il tool definitivo per il pentesting di WordPress. Iniziamo sempre con una scansione passiva (stealthy).


Cosa fa la modalità stealthy:

  • Legge solo informazioni pubbliche
  • Analizza il codice sorgente HTML
  • Controlla header HTTP e meta generator
  • Non esegue brute force di file
  • Minimo rischio di detection

API Token: Per avere info sulle CVE, registratevi su wpscan.com e usate --api-token


Security Insight: Una scansione passiva rivela spesso già la versione di WordPress, il tema e alcuni plugin. Sufficienti per una prima valutazione!

Lab: WPScan Stealthy Mode

Comando:

┌──(kali㉿kali)-[~] └─$ wpscan --url http://vulnerable-wp.local --stealthy
