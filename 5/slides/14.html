Capitolo 2: WordPress Enumeration

Slide 2.5: Theme & Timthumb Enumeration
Teoria: Anche i Temi sono Codice

I temi WordPress non sono solo grafica: contengono codice PHP eseguibile. Un tema vulnerabile è pericoloso quanto un plugin.


Vulnerabilità comuni nei temi:

  • Timthumb.php - RCE nelle versioni vecchie
  • Local File Inclusion (LFI)
  • Cross-Site Scripting (XSS)
  • Arbitrary File Upload
  • SQL Injection nei custom query

Timthumb: Libreria per il ridimensionamento immagini, afflitta da vulnerabilità RCE gravi (CVE-2011-4106).


Security Insight: Cercate sempre file timthumb.php o thumb.php. Anche se il tema è aggiornato, potrebbero essere rimasti file orfani vulnerabili.
